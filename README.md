# Patient Care Hub

A simple website prototype that helps patients combine medical records from multiple specialists and share a personal biography with care teams.

## What is included

- `index.html` — Core patient portal interface
- `styles.css` — Responsive styling for forms, cards, and layout
- `script.js` — Local data storage, record management, and biography preview

## How to use

1. Open `index.html` in your browser.
2. Add specialist records in the "Medical Records" screen.
3. Click "Merge Records" to generate a combined summary.
4. Switch to "Patient Biography" to write and save your personal information.

## Notes

- Data is stored locally in your browser using `localStorage`.
- The download button exports the merged summary as a text file.
